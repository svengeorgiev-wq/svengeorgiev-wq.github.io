# Urlaubsduell Revanche

Statische Bonus-App zur Urlaubs-Edition des Paar-Duell-Buchs.

## Zweck

- editionsgebundener Bonusbereich nur fuer die Urlaubs-Edition
- Revanche-Runden aus kuratierten Urlaubsduellen
- Ein-Handy-Spielleiter fuer zwei Personen
- Sofa-, Tisch- und Zugmodus
- digitale Punktekarte, Timer, Druckansicht und lokaler Spielstand

## Lokal starten

Die App kann direkt ueber `index.html` geoeffnet werden. Fuer eine lokale Vorschau mit Server:

```powershell
python -m http.server 4174
```

Dann oeffnen:

```text
http://localhost:4174/paar-duell-urlaub/
```
