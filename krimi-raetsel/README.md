# Krimi-Rätsel

Statische Web-App fuer leichte Sherlock-inspirierte Krimi-Raetsel.

## Repo und Slug

- Repo-Name: `Krimi-Rätsel`
- Slug: `Krimi-Rätsel`

## Lokal starten

Die App kann direkt ueber `index.html` geoeffnet werden. Fuer eine lokale Vorschau mit Server:

```powershell
python -m http.server 4173
```

Danach im Browser oeffnen:

```text
http://localhost:4173
```

## Dateien

- `index.html` enthaelt die App-Struktur.
- `styles.css` setzt den dunklen Krimi-Look passend zum Cover.
- `app.js` enthaelt die leichten Faelle, Hinweise, Loesungen und lokale Speicherung.
- `assets/cover.webp` ist das optimierte Cover des neuen Buchs.
